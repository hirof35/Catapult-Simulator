
# 空母カタパルトシミュレーター (Aircraft Carrier Catapult Takeoff Simulator)

TypeScript と HTML5 Canvas で構築された、空母戦闘機（F/A-18）の射出・発艦物理シミュレーターです。
MKS単位系（メートル・キログラム・秒）に基づいた純粋な物理演算ロジック（コア）と、ブラウザで視覚的に操作・確認できるGUIフロントエンドを疎結合に設計しています。

## 💡 本プロジェクトの狙い（設計思想）

本プロジェクトは、**「物理演算ロジック（ドメイン）」と「UI・描画（プレゼンテーション）」を完全に分離**して設計されています。
コアとなる物理エンジン（`simulator.ts`）は、HTMLやCanvasなどの特定の表示環境に一切依存していません。そのため、コンソール（CLI）アプリ、Web（GUI）アプリ、あるいはNode.jsサーバーサイドなど、あらゆる環境へそのまま移植・再利用が可能な構造（疎結合設計）になっています。

## 📊 シミュレーションの物理モデル

戦闘機を安全に発艦（テイクオフ）させるには、以下の3つの要素が物理的均衡を満たす必要があります。

1. **加速度 ($a$) の決定**: カタパルトの牽引力 $F \ [\text{N}]$ と機体質量 $m \ [\text{kg}]$ から算出。
   $$a = \frac{F}{m}$$
2. **位置・速度の離散更新**: フレームごとの微小時間 $\Delta t \ [\text{s}]$ に基づき、速度 $v$ と位置 $x$ を前進オイラー法的に更新。
   $$v_{t+\Delta t} = v_t + a \cdot \Delta t$$
   $$x_{t+\Delta t} = x_t + v_t \cdot \Delta t$$
3. **離陸（発艦）成否判定**: 90mのデッキ端に達した時点の**対気速度（Airspeed）**が、必要離陸速度（260 km/h）を満たしているかを検証。
   $$\text{Airspeed} = \text{Ground Speed (対地速度)} + \text{Headwind (向かい風)}$$

## 📂 フォルダ構成

```text
catapult-sim/
├── index.html          # 操作画面（スライダーUI、Canvas）
├── package.json         # プロジェクト構成・依存ライブラリ管理
├── tsconfig.json        # TypeScriptコンパイラ設定
├── README.md            # 本ドキュメント
└── src/
    ├── simulator.ts     # 💡 コア物理ロジック（環境依存ゼロの純粋なクラス）
    ├── main.ts          # GUI描画およびブラウザUI制御ロジック
    └── index.ts         # CLI（コンソール）環境用の動作確認テストスクリプト
🛠️ 環境構築と実行方法
1. 依存ライブラリのインストール
Node.js環境が必要になります。プロジェクトのルートで以下を実行してください。

Bash
npm install
(新規構築時は typescript, tsx, vite がインストールされます)

2. GUI（ブラウザ版）の起動
Vite高速開発サーバーを起動し、ブラウザでインタラクティブにシミュレーターを操作します。

Bash
npx vite
起動後、ターミナルに表示されるURL（例: http://localhost:5173/）にブラウザでアクセスしてください。

3. CLI（コンソール版）の起動
ブラウザを使わず、ターミナル上の数値ログだけで物理演算の挙動を追うテスト用スクリプトです。新世代実行ツール tsx を用いて高速実行します。

Bash
npx tsx src/index.ts
🎮 シミュレーターの遊び方・検証ケース
スライダーでパラメーターを動かし、航空工学・ロジスティクスの観点から発艦の成否を検証できます。

ケースA（過積載・パワー不足）
機体重量を最大（30,000kg：フル武装状態）にし、向かい風を 0 km/h にすると、デフォルトのカタパルト出力（650,000N）ではエンドまでに十分な揚力を得られず、海へ落下（CRASH）します。

ケースB（空母戦闘航海・向かい風の重要性）
ケースAの状態から、空母を全速力で走らせて強烈な「向かい風（Headwind: 50km/h以上）」を生み出すか、カタパルト出力を限界まで高めることで、同じ重量でも安全に大空へ舞い上がらせる（SUCCESS）ことが可能になります。実際の航空母艦が風上に向けて全力疾走して発艦を行う理由を、数値とアニメーションで直感的に理解できます。
